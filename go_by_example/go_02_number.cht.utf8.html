Go有很多種數據類型，包括字符串類型，整型，浮點型，布爾型等等，這裡有幾個基礎的例子。

package main

import "fmt"

func main() {

    // 字符串可以使用"+"連接
    fmt.Println("go" + "lang")

    //整型和浮點型
    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

    // 布爾型的幾種操作符
    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}

輸出結果為

golang
1+1 = 2
7.0/3.0 = 2.3333333333333335
false
true
false
