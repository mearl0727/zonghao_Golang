Go語言內置支持多返回值，這個在Go語言中用的很多，比如一個函數同時返回結果和錯誤信息。

package main

import "fmt"

// 這個函數的返回值為兩個int
func vals() (int, int) {
    return 3, 7
}

func main() {

    // 獲取函數的兩個返回值
    a, b := vals()
    fmt.Println(a)
    fmt.Println(b)

    // 如果你只對多個返回值裡面的幾個感興趣
    // 可以使用下劃線(_)來忽略其他的返回值
    _, c := vals()
    fmt.Println(c)
}

輸出結果為

3
7
7
