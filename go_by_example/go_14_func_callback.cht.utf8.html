Go支持函數回調，你可以把函數名稱作為參數傳遞給另外一個函數，然後在別的地方實現這個函數。

package main

import "fmt"

type Callback func(x, y int) int

func main() {
    x, y := 1, 2
    fmt.Println(test(x, y, add))
}

//提供一個接口，讓外部去實現
func test(x, y int, callback Callback) int {
    return callback(x, y)
}

func add(x, y int) int {
    return x + y
}

運行結果

3
