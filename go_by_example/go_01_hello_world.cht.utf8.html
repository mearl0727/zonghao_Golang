我們的第一個例子是打印經典的「hello world」信息，我們先看下代碼。

package main

import "fmt"

func main() {
	fmt.Println("hello world")
}

輸出結果為：

hello world

為了使一個go文件能夠編譯為可執行文件，包名必須是main，然後我們導入提供格式化輸出的fmt包，該程序的執行入口是func main()函數，在函數裡面，我們使用fmt包提供的Println函數來輸出"hello world"字符串。

為了運行這個程序，我們可以使用go run hello_world.go來運行這個例子，這樣是直接輸出運行結果而不會產生任何中間文件。但是有的時候我們希望能夠將程序編譯為二進制文件保存起來，我們可以像上面一樣使用go build hello_world.go來將源代碼編譯為二進制可執行文件。然後我們可以直接運行這個二進制可執行文件。

好了，第一個例子就這樣結束了。很簡單。
