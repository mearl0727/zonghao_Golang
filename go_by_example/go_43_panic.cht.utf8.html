Panic表示的意思就是有些意想不到的錯誤發生了。通常我們用來表示程序正常運行過程中不應該出現的，或者我們沒有處理好的錯誤。

package main

import "os"

func main() {

    // 我們使用panic來檢查預期不到的錯誤
    panic("a problem")

    // Panic的通常使用方法就是如果一個函數
    // 返回一個我們不知道怎麼處理的錯誤的
    // 時候，直接終止執行。
    _, err := os.Create("/tmp/file")
    if err != nil {
        panic(err)
    }
}

和其他的編程語言不同的是，Go並不使用exception來處理錯誤，而是通過函數返回值返回錯誤代碼。
